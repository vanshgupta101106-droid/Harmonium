# iHarmonium 🎹

A digital harmonium instrument controlled using the **laptop lid sensor**.

## Features

* Laptop lid movement acts like harmonium bellows
* Keyboard keys play musical notes
* Real-time WebSocket communication
* Web-based harmonium interface

## Technologies

* HTML
* JavaScript
* Python
* WebSockets

## How to Run

1. Start the Python bridge

```
python harmonium.py
```

2. Open the web app

```
harmonium.html
```

3. Press keyboard keys to play notes and move the laptop lid to pump air.
